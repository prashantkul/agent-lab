Student Help Guide

Everything you need to know about completing your coursework

Quick Start

  1. 1 Access your assigned module from the dashboard
  2. 2 Study the PDF course material
  3. 3 Accept and complete the GitHub Classroom assignment
  4. 4 Submit your work and view your grade

Your enrolled modules appear on your dashboard based on the course schedule.

Each module includes:

  • PDF Course Material - Lecture slides and reference material
  • Assignment Overview - Summary of what you'll build
  • GitHub Assignment - Hands-on coding exercise

Modules unlock weekly according to the course schedule.

Step 1: Accept the Assignment

Click the green Accept Assignment button. This creates your personal repository.

Step 2: Clone Your Repository

git clone https://github.com/your-org/assignment-your-username.git
cd assignment-your-username

Step 3: Complete the Assignment

Follow the instructions in the README.md file. The assignment will guide you through building an AI agent.

Step 4: Test Locally

# Install dependencies
pip install -r requirements.txt

# Run your code
python main.py

Step 5: Submit Your Work

git add .
git commit -m "Complete assignment"
git push

Step 6: Check Your Grade

After pushing, go to the Actions tab in your GitHub repository to see the autograder results.

Your assignments are automatically graded when you push to GitHub.

Grading criteria typically includes:

  • Code structure and organization
  • Correct implementation of required features
  • Passing test cases
  • Code quality and best practices

To view detailed results:

  1. Go to your repository on GitHub
  2. Click the "Actions" tab
  3. Click on the latest workflow run
  4. Expand the grading step to see detailed feedback

Tip: You can push multiple times to improve your grade before the deadline!

If you're stuck on an assignment:

  • Re-read the assignment README carefully
  • Check the course PDF for relevant concepts
  • Look at the test cases for hints on expected behavior
  • Review the error messages from the autograder

If you need more help:

  • Post in the course discussion forum
  • Attend office hours
  • Email the instructor or TA

Still have questions?

Contact your instructor or post in the course forum.
